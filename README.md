# 画像収集ソフト

キーワードを指定して画像を自動収集するWebアプリケーションです。

## 機能

- キーワードを入力して画像を検索・収集
- 保存先フォルダの指定
- 最大画像数の設定
- 最小画像サイズの指定（オプション）
- 収集した画像のプレビュー表示

## セットアップ

### 1. 必要なパッケージのインストール

```bash
pip install -r requirements.txt
```

### 2. アプリケーションの起動

```bash
streamlit run app.py
```

ブラウザが自動的に開き、`http://localhost:8501` でアプリケーションが表示されます。

## 使い方

1. サイドバーで検索キーワードを入力
2. 保存先フォルダ名を指定
3. 最大画像数を設定
4. （オプション）最小画像サイズを指定
5. 「画像収集を開始」ボタンをクリック

## 技術スタック

- **Streamlit**: Webアプリケーションフレームワーク
- **icrawler**: 画像クローラーライブラリ
- **BingImageCrawler**: Bing画像検索を使用
